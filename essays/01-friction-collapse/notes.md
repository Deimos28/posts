# The friction collapse
Published: https://deimos28.substack.com/p/the-friction-collapse (p-204088857)

Essay 1 of the series. Core claim: for the first time, the abstracted layer
(judgement) has no oracle. Friction generated judgement as a side effect of
production; agents sever the bundle.

Closing line: "The question is not whether human judgement will still be needed in
ten years: it is whether we will still be able to judge."
