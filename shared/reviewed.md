# Reviewed

Ideas that have been metabolised. An entry belongs here only when the author has
reconstructed the argument himself — not when an agent's version has been approved.
Entries are written by the author. Agents do not write to this file.

Essays may cite this file. Essays may not cite the scratchbook.
