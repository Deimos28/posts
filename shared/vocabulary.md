# Vocabulary (canonical)

One line per term. If an essay needs a term to mean something different, this file
changes first, in its own commit. Terms marked (external) are cited, not coined here.

- **Friction collapse** — agents severing the bundle by which comprehension arrived
  as a side effect of production. (Essay 1.)
- **Cognitive debt** (external: Kosmyna et al., arXiv 2506.08872; also Storey) — the
  gap between what you shipped and what you understand; deferring mental effort at
  long-term cost.
- **Comprehension cursor** — the choice, per task, of where to remain falsifiable.
  Paying means staking expectations specific enough for reality to violate; borrowing
  means obtaining the result without exposing your own model to refutation.
  Previously pinned at "pay"; agents unpinned it.
- **Friction transducer** — the component that converted the act of manual coding
  into measurements (ŷ) consumed by comprehension. Removed on the agent path.
- **Leaky reservoir / top-up** — comprehension decays regardless; manual coding was a
  continuous, default-on refresh (accidental spaced repetition). Agents stop the
  top-up, not start the decay.
- **Rated mental model** — the model plus a confidence rating; the only thing
  judgement consumes. Everything that measures feeds comprehension; judgement only
  ever sees the model.
- **The alarm** — fear/unease as prediction error (expected − observed), where
  "expected" is generated by comprehension. Self-masking: the detector dies with the
  thing it detects.
- **Becoming vs doing knowledge** — understanding whose acquisition transforms you
  (needs friction) vs understanding you merely use (accelerate freely).
- **Blast radius** — expected cost of a wrong answer in a given region; the location
  rule for the cursor and the ranking criterion for interrogation and compression.
- **3X** (external: Kent Beck) — Explore/Expand/Extract; the phase rule for the
  cursor.
- **Dependency inversion (of the teacher)** — take structure/exploration from the
  LLM, do the substance (words, code) yourself. Companion, not substitute. Purpose is
  understanding, not speed.
- **The conveyor belt** — each generation's substance becomes the next generation's
  mechanical layer (register allocation → compiler; boilerplate → agent). The
  narrative layer stays off the belt because it has no oracle.
- **Predict-before-reveal** — formulate the prediction before seeing the agent's
  output; detector maintenance, keeps the expected term sharp enough to be violable.
- **Scheduled interrogation** (was: Inquisition) — recurring private quiz on the
  standing system; the only continuous observation channel for cognitive debt.
- **Speculative-change probes** (was: flexer army) — implement hypothetical future
  requirements to measure cost-of-change; affordable only since agents.
- **The canary** (was: re-injection) — replay known faults through the detection
  funnel; distinguishes "no debt found" from "debt-finding is broken".
- **The funnel** — triage architecture for agent-scale review volume: classifier
  ranks/escalates a thin slice to human attention; manual unease channel kept for
  uncorrelated errors.
- **Small, Clear, Early, Surgical** — bounded reviewable units; prompt-as-spec (the
  prompt doubles as the prediction); fast loop closure; attention concentrated where
  blast radius is high.
- **Blind-faith strategy** — generate-test-regenerate against an oracle, acquiring no
  comprehension. Exactly as viable as the oracle is fast; legitimate on the verifiable
  layer.
- **No-oracle layer** — the part of the system with no fast, complete verifier
  (architecture, security properties, slow rot). Where comprehension spend
  concentrates; where the fitness program lives.
- **Two currencies** — human attention (appreciates with use; mints understanding)
  vs LLM tokens (constant quality; pure outflow). Non-fungible while models don't
  learn from your system.
- **The concession** — engineering does not decide; the business holds allocation and
  sees only behavioural signals. The engineer's one lever is the latency of the debt
  signal.
- **Falsifiability boundary** — what the cursor sets. On one side the engineer stakes
  claims reality can refute; on the other he borrows outcomes and cannot be wrong.
  Naur (the program is a theory) + Popper (a theory is knowledge only insofar as it is
  falsifiable): borrowed comprehension is theory that risks nothing.
- **Calibration event** — any moment where a specific expectation meets reality and can
  update the rated mental model. Friction is one implementation among several
  (debugging, incidents, reading, probes); manual coding supplied them without being
  asked.
- **Friction** — the cost of exposing a model to reality: effort, time, discomfort, and
  the risk of being proven wrong.
